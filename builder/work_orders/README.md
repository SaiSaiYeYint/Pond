# Work Orders

This folder contains Grimm-approved work orders for AI builders.

During Work Time, Grimm should review unfinished work orders one at a time or in small groups.

## Unfinished

A work order is unfinished when its status is:

- `draft`
- `approved`
- `building`

## Finished

A work order is finished when its status is:

- `completed`
- `cancelled`

## Review Rules

Grimm may:

- summarize work orders
- reorder priority
- merge duplicates
- mark work orders completed
- archive cancelled work orders

Grimm must not execute the work order during review.

Any AI builder must read `BUILD.md`, `grimm/constitution.md`, and all unfinished work orders before coding.
