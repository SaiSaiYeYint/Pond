# Grimm App Static Prototype

This folder is now set up for a simple static upload.

Upload these files/folders to GitHub:

- `index.html`
- `app.js`
- `assets/`
- `grimm/`
- `data/`
- `README.md`
- `.gitignore`

The app currently runs without API or backend code. Grimm uses local scripted personality, local judging, local chat memory, pond visuals, and browser storage.

For Vercel, import the GitHub repo as a static site. No environment variables are needed for this version.
